# JShell Anleitung
1. Starten Sie die  Konsole
2. In der Console eingeben ("jshell.exe --class-path .\out\production\P03-PokePong")
3. Dann import PokePong.model.PongModel
4. Ein Object erstellen ("var Model=new PongModel(300,300)"). Geben Sie in der Klammern sollte Zahlen für Spielfeld geben.
5. Um Ball position zu ändern geben Sie ("Model.data.getBall().getPosition().setY") oder ("Model.data.getBall().getPosition().setX")benutzen.
6. Für player können Sie ("Model.data.getPlayer1().getPosition.setX(100)") und ("Model.data.getPlayer1().getPosition.setY(100)") für spieler 2 verändern Sie getPlayer2.
7. Um ausgabe zu bekommen, geben Sie ("Model.toString") ein
